# Web-Directory-Scanner
A Python tool that scans websites for hidden directories using a wordlist. Built for educational purposes and authorised penetration testing practice only.
## What It Does
- Takes a target URL as input
- Reads from a wordlist file
- Makes HTTP requests to each potential directory
- Reports directories that return a 200 status code

## Legal Notice
This tool is for educational purposes only.
Only use against websites you own or have explicit 
written permission to test. Unauthorised scanning 
is illegal.
